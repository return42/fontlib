Config¶

Fontlib can be configured with a configuration file and/or with command line arguments (see Fontlib Options).

# -*- coding: utf-8; mode: ini -*-

[DEFAULT]

workspace = ~/.fontlib

[fontstack]

# Fonts loaded from builtins.
builtin fonts = cantarell, dejavu

# Fonts loaded from entry points.
entry points = fonts_ttf, fonts_otf, fonts_woff, fonts_woff2

# Full qualified name of the URL-cache implementation
cache = fontlib.urlcache.URLCache

[google fonts]

# Select font families from https://fonts.google.com/
family base url = https://fonts.googleapis.com/css?family=
fonts = Leckerli One, Roboto Slab

[logging]

# Threshold for the logger
# Value: debug | info | warning | error | critical
level = warning
#config = %(workspace)s/log.ini

# -*- coding: utf-8; mode: ini -*-
#
# fontlib logging configuration
#
# https://docs.python.org/3/library/logging.config.html#configuration-file-format
#

[loggers]
keys = root, fontlib

[logger_root]
level = NOTSET
handlers = null

[logger_fontlib]
level = WARNING
handlers = console
#handlers = console, logfile

# The qualname entry is the hierarchical channel name of the logger, that is to
# say the name used by the application to get the logger.
qualname = fontlib
propagate = 1

[handlers]
keys = null, console, logfile

[handler_null]
class = NullHandler
level = NOTSET
args = ()

[handler_console]
class = StreamHandler
level = DEBUG
formatter = console
# https://docs.python.org/3.7/library/logging.handlers.html#logging.StreamHandler
args = (sys.stderr, )

[handler_logfile]
class = handlers.RotatingFileHandler
level = DEBUG
formatter = logfile
# https://docs.python.org/3.7/library/logging.handlers.html#logging.handlers.RotatingFileHandler
args = ('%(WORKSPACE)s/%(APP)s.log', )
kwargs = {'maxBytes' : 1024 * 1024, 'backupCount' : 10 }

[formatters]
keys = console, logfile

[formatter_console]
format = %(levelname)-8s|%(name)-12s| %(message)s

[formatter_logfile]
format = %(asctime)s| %(levelname)-8s|%(name)-12s| %(message)s

basic config and logging setup¶

To install templates for indiviual configuration use the config install command:

$ fontlib config install ~/.fontlib
using workspace: /home/markus/.fontlib
set log level of logger: fontlib (WARNING)
log goes to:
 - <StreamHandler <stderr> (DEBUG)>
install:  ~/.fontlib/config.ini
install:  ~/.fontlib/log.ini

This will setup a workspace folder .fontlib in your home folder where templates for application defaults and logging defaults are copied in. To activate the configuration in .fontlib/log.ini just un-comment the config line:

[logging]
...
config = %(workspace)s/log.ini

inspect configuration¶

With command fontlib config show you can inspect current configuration options. To show your current configuration use fontlib --verbose config show. With switch --verbose you will get some more information.

$ ../local/py3/bin/fontlib --verbose config show
load configuration from: /home/markus/.fontlib/config.ini

config.ini
==========

[DEFAULT]
workspace = /home/markus/.fontlib

[fontstack]
builtin fonts = cantarell, dejavu

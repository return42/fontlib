Use¶

The fontlib command¶

Command line tools from the fontlib library.

The fontlib library comes with a command line, use fontlib --help for a basic help. For a detailed help to one of the subcommands type fontlib <command> –help``.

usage: fontlib [-h] [--debug] [--verbose] [--quiet] [-c [INI-FILE]]
               [--workspace WORKSPACE]
               {README,version,list,css-parse,download,config} ...

Command line tools from the fontlib library.

optional arguments:
  -h, --help            show this help message and exit
  --debug               run in debug mode (default: False)
  --verbose             run in verbose mode (default: False)
  --quiet               run in quiet mode (default: False)
  -c [INI-FILE], --config [INI-FILE]
                        specify config file (default:
                        /home/markus/.fontlib/config.ini)
  --workspace WORKSPACE
                        workspace (default: ~/.fontlib)

commands:
  {README,version,list,css-parse,download,config}
    README              prints README to stdout
    version             prints version infos to stdout
    list                List fonts from *builtins*, *fonts.googleapis.com* and
                        other resources.
    css-parse           Parse ``@font-face`` rules from <url>.
    download            Download font-family <family> into folder <dest>.
    config              Inspect configuration (working with INI files).

Fontlib Options¶

  • --verbose: more verbose output

  • -c / --config: gitfontlib/config.ini (see Config)

  • --workspace: place where application persists its data

Fontstack options¶

  • --builtins: select Builtin fonts

  • --ep-fonts: Fonts from entry points

  • --google: Fonts from google

fontlib list¶

usage: fontlib list [-h] [--builtins [BUILTINS]] [--ep-fonts [EPFONTS]]
                    [--google [GOOGLE]]

optional arguments:
  -h, --help            show this help message and exit
  --builtins [BUILTINS]
                        use builtin fonts (default: cantarell, dejavu)
  --ep-fonts [EPFONTS]  use fonts from entry points (default: fonts_ttf,
                        fonts_otf, fonts_woff, fonts_woff2)
  --google [GOOGLE]     use fonts from fonts.googleapis.com (default: Leckerli
                        One, Roboto Slab)

List fonts from *builtins*, *fonts.googleapis.com* and other resources.

Option --google add fonts from google fonts infrastructure.  Select font
families from:

    https://fonts.google.com

Option --ep-fonts add fonts from entry points, see:

    https://pypi.org/project/fonts

To install fonts from the *python fonts project* e.g. run::

    $ pip install --user \
          font-amatic-sc font-caladea font-font-awesome \
          font-fredoka-one font-hanken-grotesk font-intuitive \
          font-source-sans-pro font-source-serif-pro

fontlib css-parse¶

usage: fontlib css-parse [-h] url

positional arguments:
  url         URL of stylesheet (CSS) to parse

optional arguments:
  -h, --help  show this help message and exit

Parse ``@font-face`` rules from <url>.

Load CSS (stylesheet) from <url> and filter ``@font-face`` rules.  E.g. google
fonts API 'https://fonts.googleapis.com/css?family=Cute+Font|Roboto+Slab' or
built-in *dejavu* fonts from url 'file:./fontlib/files/dejavu/dejavu.css'.

fontlib download¶

usage: fontlib download [-h] [--builtins [BUILTINS]] [--ep-fonts [EPFONTS]]
                        [--google [GOOGLE]]
                        dest family [family ...]

positional arguments:
  dest                  Folder where the download will be placed.
  family                Font's name, the value of CSS font-family
                        property.E.g. 'Roboto Slab' or 'DejaVu Sans Mono'

optional arguments:
  -h, --help            show this help message and exit
  --builtins [BUILTINS]
                        use builtin fonts (default: cantarell, dejavu)
  --ep-fonts [EPFONTS]  use fonts from entry points (default: fonts_ttf,
                        fonts_otf, fonts_woff, fonts_woff2)
  --google [GOOGLE]     use fonts from fonts.googleapis.com (default: Leckerli
                        One, Roboto Slab)

Download font-family <family> into folder <dest>.

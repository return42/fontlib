The python fontlib package helps to manage fonts from different resources. It comes with an API and the fontlib command line (see usage).

To name just a few fontlib features:

  • fontlib ships some builtin fonts

  • fontlib make use of fonts from google fonts

  • fontlib use fonts from entry points

  • Python mimetypes for font types

Usage¶

The python fontlib package helps to manage fonts from different resources. A Font itself is identified by its origin URL. Fonts from various resources (URLs) are organized in a FontStack. You can setup a FontStack individual, but for the first the fontlib command has some handy defaults for you. Jump directly to chapter get started to get in touch with fontlib’s command line.
